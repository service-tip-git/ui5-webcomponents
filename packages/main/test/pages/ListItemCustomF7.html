F7/F2 Key Test

F7 vs F2 Behavior:

  • F2: Simple navigation - always goes to first focusable element
  • F7: Smart navigation - remembers last focused element position across items

Test Steps:

  1. Click on first list item
  2. Press F7 → should go to first button
  3. Press TAB to move to second button
  4. Press F7 → should return to list item
  5. Press ArrowDown → should go to second list item
  6. Press F7 → should go to SECOND button (maintains position!)
  7. Test F2 → should always go to first button (no memory)
